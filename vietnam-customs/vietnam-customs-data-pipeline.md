# 越南海关橡胶数据抓取流程

## 概述

从越南海关总署（customs.gov.vn）下载月度统计PDF，提取橡胶（Cao su）和橡胶制品（Sản phẩm từ cao su）数据，写入飞书多维表格。

**目标数据**：每月4份PDF × 12月 = 48份

| 类型代码 | 含义 | 提取内容 |
|---------|------|---------|
| 2x | 出口商品明细 | 橡胶（数量+金额+同比）、橡胶制品（金额+同比） |
| 2n | 进口商品明细 | 同上 |
| 5x | 出口国别明细 | 每国橡胶（数量+金额）、橡胶制品（金额） |
| 5n | 进口国别明细 | 同上 |

## 飞书表格

- **App Token**: 环境变量 `FEISHU_APP_TOKEN`
- **商品汇总表**: 月份、方向、品类、当月数量(吨)、当月金额(万美元)、累计数量(吨)、累计金额(万美元)、同比(%)
- **国别明细表**: 月份、方向、国家(中文)、国家(越南语)、品类、当月金额(万美元)、累计金额(万美元)、当月数量(吨)、累计数量(吨)
- **重点追踪表**: 同国别表字段，仅记录 橡胶→中国(出口) + 橡胶制品→美国(出口)

## PDF URL 规律

```
https://files.customs.gov.vn/CustomsCMS/TONG_CUC/{发布年}/{月}/{日}/2025-t{月份}-{类型}(vn-sb).pdf
```

- 发布时间：次月第一周（通常5-8号）
- 12月数据可能跨年发布（如2026/1/7），5n可能更晚（如2026/3/9）
- 类型：2x/2n/5x/5n，月份用数字（t1=1月，t12=12月）

### 完整URL构造规则

对于第M月的数据，发布年月按以下规律：
- M月数据 → 次月(M+1)发布，路径中用发布时间
- 例外：12月数据在次年1月发布，5n可能延迟到3月

**快速构造方法**：已知同月其他类型PDF的URL，替换类型代码即可。

## 执行流程

### Step 1: 下载PDF

```bash
# 按URL规律逐个下载，用curl -sI验证（200且content-length>1KB才算有效）
# 保存到 {VIETNAM_DATA_DIR}/{年}/{月}/
```

**注意事项**：
- 网站是JS渲染的，直接fetch拿不到链接，需从已知URL模式构造
- 0字节文件说明URL不对或网络超时，换日期重试
- 用 `curl -sI` 检查HTTP状态和content-length

### Step 2: PDF → JSON（pdftext）

```bash
PDFTOOL="{PDFTOOL_PATH}"
$PDFTOOL --sort --json input.pdf > output.json
```

JSON结构：`pages[i].blocks[j].lines[k].spans[l].text`

### Step 3: 解析JSON

**解析脚本**: `parse_vietnam.py`

关键解析逻辑：
- **2x/2n（商品汇总）**: 用正则匹配 `Cao su Tấn` 和 `Sản phẩm từ cao su USD` 行
- **5x/5n（国别明细）**: 先按全大写越南语国名分块，再在每块内搜索橡胶行
  - 国家检测：正则匹配全大写越南语行，排除商品关键词
  - 橡胶行：`Cao su Tấn {当月数量} {当月金额} {累计数量} {累计金额}`
  - 制品行：`Sản phẩm từ cao su USD {当月金额} {累计金额}`

**越南语国家名映射**（部分）:
```
TRUNG QUỐC=中国, HOA KỲ=美国, NHẬT BẢN=日本, HÀN QUỐC=韩国,
ẤN ĐỘ=印度, THÁI LAN=泰国, MALAIXIA=马来西亚, INĐÔNÊXIA=印尼,
ĐÀI LOAN=台湾, SINGAPO=新加坡, ANH=英国, ĐỨC=德国,
PHÁP=法国, HÀ LAN=荷兰, BỈ=比利时, ITALIA=意大利,
CAMPUCHIA=柬埔寨, PHILIPPIN=菲律宾, NGA=尼日利亚
```

**数值格式**: 越南语用`.`作千分位，`,`作小数点。解析时先去掉`.`再替换`,`为`.`。

### Step 4: 转换 & 写入

**金额转换**: 原始值为USD，需除以10000转为万美元。

**写入方案**（优先级从高到低）:
1. **飞书API batch_create**: 每批≤50条（参数大小限制），适合小批量
2. **CSV导出+手动导入**: 适合大批量（>100条），UTF-8 BOM编码，通过飞书DM发送
3. API写入大批量数据效率低，子代理也不适合此类重复工具调用

**CSV字段顺序**: 月份,方向,国家(中文),国家(越南语),品类,当月金额(万美元),累计金额(万美元),当月数量(吨),累计数量(吨)

## 已知坑点

1. **PDF缺失不等于真缺失**: 网站JS渲染导致抓取不到链接，但PDF可能存在。用URL模式构造+curl验证可找到"隐藏"的PDF
2. **0字节文件**: 可能是网络问题，不是URL不对，重试即可
3. **国家名识别不全**: COUNTRY_MAP只覆盖了常见国家，其他国家直接保留越南语原名
4. **"Cao su nguyên"干扰**: 搜索"Cao su"时需排除含"nguyên"的行（天然橡胶原料）
5. **API参数大小限制**: 50条JSON约7.6KB安全，100条约17KB会超限
6. **飞书删除需授权**: `base:record:delete` 权限需要用户授权才能操作

## 后续复用

抓取新年份数据时：
1. 更新URL中的年份（如2025→2026）和月份编号
2. 注意12月数据可能跨年发布
3. 复用 parse_vietnam.py 解析脚本
4. 国别映射表可能需要补充新出现的国家
